## 说明

本站由 [Docsify](https://docsify.js.org/) 驱动，无需本地构建；将本仓库推送到 GitHub 并开启 **GitHub Pages** 后即可访问。

## 编辑与样式

- 新增报告、侧栏链接与手机访问说明：见 [如何更新站点](guide.md)。
- 报告内若要用 **关键词 / 标签 / 比例条 / 地区 chip** 等 HTML 片段：**完整可复制示例**见 [指南 → 附录](guide.md#cba-html-appendix)。

访问量与国家分布需在 `index.html` 的 `CBA_SITE_CONFIG` 中配置 Plausible 或 GA4 后在后台查看；详见 [guide.md](guide.md)。
